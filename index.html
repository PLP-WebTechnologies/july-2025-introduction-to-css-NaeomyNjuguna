About CSS

CSS (Cascading Style Sheets) allows you to style and design web pages. It controls colors, layouts, fonts, spacing, and overall appearance.

The CSS Box Model

Every HTML element is treated as a box. The box model consists of:

  • Content: The text or image inside the box.
  • Padding: Space between content and border.
  • Border: Surrounds the padding and content.
  • Margin: Space outside the border.
